# Journal and Mood Tracking Application

## Overview
This application allows users to track their daily moods and maintain a journal. It aims to promote mental well-being by enabling self-reflection and mood analysis.

## Features
- **Mood Tracking**: Users can select their mood from predefined options or enter custom moods.
- **Journaling**: Users can write daily entries reflecting their thoughts and feelings.
- **Analytics**: The app provides insights into mood patterns over time, helping users identify triggers and trends.

## Technologies Used
- Frontend: React
- Backend: Node.js with Express
- Database: MongoDB

## Setup
1. Clone the repository: `git clone https://github.com/Larrydesignn/see`
2. Navigate to the project directory: `cd see`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
